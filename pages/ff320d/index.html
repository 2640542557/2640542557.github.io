centos安装docker

参考官方文档 https://docs.docker.com/engine/install/centos/ (opens new window)

Docker教程 https://www.runoob.com/docker/docker-tutorial.html (opens new window)

# 1、卸载旧版本

yum remove docker \
           docker-client \
           docker-client-latest \
           docker-common \
           docker-latest \
           docker-latest-logrotate \
           docker-logrotate \
           docker-engine

# 2、需要的安装包

yum install -y yum-utils

# 3、设置镜像的仓库

# 使用阿里国内源安装docker，官方的太慢
yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

# 更新yum软件包索引（centos7）
yum makecache fast 

# 更新yum软件包索引（centos8没有fast命令，去掉即可）
yum makecache

# 4、自定义配置

# 下载自定义配置相关
wget -P /etc/docker/ https://gitee.com/hc535721622/myimage/raw/be6d4eabd6d4ac86e873e1be252440963ac27370/docker/daemon.json

# 查看配置文件
[root@ecs ~]# cat /etc/docker/daemon.json

{
  "registry-mirrors": ["https://92u4ljs3.mirror.aliyuncs.com"],
  "data-root": "/data/docker",
  "log-driver":"json-file",
  "log-opts": {"max-size":"500m", "max-file":"3"}
}

daemon.json配置文件说明

  • registry-mirrors：镜像加速
  • data-root：Docker运行时使用的根路径，默认/var/lib/docker
  • log-driver：容器日志的默认驱动程序（默认为“ json-file”）
  • log-opts：容器默认日志驱动程序选项，目的是限制日志文件个数及日志文件大小，避免出现日志无限增大导致磁盘满了的问题。

# 5、安装docker

因为centos8默认使用podman代替docker，直接安装docker会产生冲突，因此：

# 先移除podman
$ yum erase podman buildah

# 安装docker
$ yum install docker-ce docker-ce-cli containerd.io -y

# 6、启动docker

systemctl start docker
# 设置开机自动启动
systemctl enable docker  

# 加载重启docker
systemctl daemon-reload
systemctl restart docker

# 7、查看是否成功

docker info
  • 显示如下信息，就表示安装成功了。

# 8、其他说明：安装特定版本

参考链接：https://docs.docker.com/engine/install/centos/ (opens new window)

# 要安装特定版本，请首先列出存储库中的可用版本：
yum list docker-ce --showduplicates | sort -r

返回的列表取决于启用的存储库，并且是特定的 到你的 CentOS 版本（在本例中由后缀指示）。.el7 通过其完全限定的软件包名称安装特定版本，即 包名称 （） 加上版本字符串（第 2 列） 从第一个冒号 （） 开始，直到第一个连字符，分隔为 连字符 （）。例如。docker-ce:-docker-ce-23.0.1 替换为所需的版本，然后运行以下命令进行安装：<VERSION_STRING>

sudo yum install docker-ce-<VERSION_STRING> docker-ce-cli-<VERSION_STRING> containerd.io docker-buildx-plugin docker-compose-plugin

2023-02-23最新版本：23.0.1

yum install docker-ce-23.0.1 docker-ce-cli-23.0.1 containerd.io docker-buildx-plugin docker-compose-plugin -y
