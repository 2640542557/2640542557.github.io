    # 1. ps：Process Status 进程状态

    主要选项有：

    -e —– 显示所有进程
-f —— 全格式
-h —– 不显示标题
-l —— 长格式
-w —– 宽输出
a —— 显示终端上的所有进程，包括其他用户的进程
r ——- 只显示正在运行的进程
u —— 以用户为主的格式来显示程序状况
x —— 显示所有程序，不以终端机来区分

    查询运行中的程序进程id

    ps -ef|grep prepose-4.39.jar

    # 3. .sh 脚本添加执行权限

    ==为所有 .sh 脚本添加执行权限== chmod u+x *.sh，表示对当前目录下的file.sh文件的所有者增加可执行权限。

    u 代表所有者；
x 代表执行权限；
+ 表示增加权限;
*.sh 表示当前目录下所有 .sh 文件。

    ==为指定 .sh 脚本添加执行权限== chmod u+x file1.sh，表示对当前目录下的 file1.sh 文件的所有者增加可执行权限。

    u 代表所有者；
x 代表执行权限；
+ 表示增加权限；
file1 表示指定的脚本文件。

    # 7. tar命令

    打包范例：将整个 test 目录下的文件全部打包成为 test.tar

    # 仅打包，不压缩！
[huang@localhost shell]$ tar -cvf test.tar test
# 打包后，以  gzip  压缩
[huang@localhost shell]$ tar -zcvf test.tar.gz test

    解压文件

    # 解压tar文件
[huang@localhost shell]$ tar -xvf test.tar
# 解压gz文件
[huang@localhost shell]$ tar -zxvf test.tar.gz

    列出压缩文件内容

    [huang@localhost shell]$ tar -tvf test.tar
[huang@localhost shell]$ tar -tzvf test.tar.gz

    # 9.查看目录和文件大小: du -sh

    1. 对当前目录下所有的目录和文件的大小进行汇总,-s表示汇总,-h表示以KB, MB, GB, TB格式进行人性化显示
    du -sh .

    2. 对当前目录下每一个目录和文件的大小分别进行汇总

    du -sh *
